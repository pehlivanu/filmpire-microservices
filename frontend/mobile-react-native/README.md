# Filmpire Mobile Application (React Native)

Cross-platform mobile application for Filmpire built with React Native 0.76.3 and Expo SDK 52.

## Tech Stack

- **React Native**: 0.76.3
- **Expo SDK**: 52.0.0
- **React**: 19.0.2
- **TypeScript**: 5.7.2
- **React Native Paper**: 5.12.5

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Building for Production

```bash
# Build Android APK
npm run build:android

# Build iOS app
npm run build:ios
```

## Project Structure

```
mobile-react-native/
├── app/              # Expo Router pages
├── components/       # React Native components
├── services/         # API services
├── store/           # Zustand stores
└── types/           # TypeScript types
```
